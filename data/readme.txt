Data :

1. db_random_naive.csv - data of 415 subjects from 2 naive experiments and the random experiment.
2. db_all.csv - data of all naive and behavioral experiments.

Columns :
advice - a vector of all recommendations during the game
reward - final reward
choices - a vector of all user picks during the game
results - a vector of all the values the user got during the game from the machines
timeToReadInstructions - time in seconds to read the instructions
timeToTakeQuiz - time in seconds to take the quiz
quizWrongTriesCounter - number of times user didn't pass successfully the quiz
answerTimePerRound - a vector of time in seconds for each round (measured from the start of the round until the end of the round)
